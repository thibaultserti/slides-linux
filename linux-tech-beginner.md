## Quelques commandes pour commencer ...

```console [1-2|3-4|5|6-7]
$ pwd
/home/tibo/tmp
$ ls
test1 test2 toto/
$ cd toto/
$ pwd
/home/tibo/tmp/toto
```

* `pwd` permet d'afficher le répertoire courant
* `ls` permet de lister les fichiers et les répertoires d'un dossier
* `cd` permet de changer de répertoire
